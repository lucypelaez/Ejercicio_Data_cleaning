# Ejercicio_Data_cleaning. (Píldora Eva)

Ejercicio de limpieza de datos correspondiente a la Píldora de Eva, en el que se realiza un proceso completo de limpieza de datos de un data set sencillo de un gimnasio paso por paso con Jupiter Notebook.

## 🐼 Práctica de Data cleaning (pandas)

**Cómo usar este cuaderno:**

***Guía práctica de pasos a seguir:***

* Completa el código en cada reto.
* Ejecuta el check si existe.
* Si algo falla, corrige y vuelve a intentar, ¡sin miedo a destruir !
🔥 ¡Vamos a ello!

## 🏋️ El gimnasio de Javi

Javi lleva meses registrando sus entrenamientos en un archivo. El problema: lo hizo sin ningún orden. 😅

Fechas raras, datos que faltan, valores imposibles, duplicados...

Nuestra misión: limpiar el dataset para que se pueda analizar correctamente. 💪

**¿Qué vamos a practicar?**

**Paso a paso** 

1	Cargar un CSV con pandas

2 Explorar y entender los datos

3	Detectar y eliminar duplicados

4	Manejar valores nulos

5	Limpiar y convertir fechas

6	Detectar y corregir valores erróneos

7	Análisis final

🌱 Nivel: Principiantes del bootcamp

⏱️ Tiempo estimado: 50 minutos
